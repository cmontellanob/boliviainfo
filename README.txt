Requires at least: 4.9.X
Requires PHP: 7.0
El Plug in sirve para incorporar información de Bolivia  a un sitio creado con Wordpress, tiene 2 widgets, uno de información del país como el cambio de moneda, el clima y noticias ,otro para mostrar post de recetas de comidas, maneja post recetas y permite  mostrar galería de imágenes en efecto iluminado, funciona en ingles y español
== Description ==
El Plug info Bolivia cuanta con 2 widgets
1.- Sirve para incorporar información de Bolivia  de la cotización de dólar que extrae directamente del Banco Central de Bolivia, la temperatura actual obtenida desde un servidor web externo y las noticias directamente del sitio de la agencia ABI, este widget permite configurar el numero de noticias que se mostrara, la ciudad de la cual se mostrar el clima y si se mostrara las temperaturas máxima y mínima.
2.- y  a un sitio creado con Wordpress, tiene 2 widgets, uno de información del país, otro para mostrar las recetas, maneja post recetas y permite  galería de imágenes  además que funciona en ingles y español
Incorpora un nuevo tipo de post denominado recetas, con campos especiales como ser ingredientes ,  modo de preparación es picante condimentos y acompañantes, para que el usuario pueda tener acceso a estos post se relaciono con un widget.
Al mostrar un post determina si es del tipo galería, de ser así muestra la galería de fotos mediante un efecto que permite ver las imágenes de tamaño grande y reccorrerlos uno por  uno .
Se ha incorporado después de cada post vea también con enlaces relacionados a otros post.
== Installation ==
Para Instalar solamente copie el contenido de la carpeta en wp-content\plugings ,
Ingrese al administrador de wordpress  actívelo
notara que se han incrementado 2 plug ing para incoporararlos, además que si usted crea un post de galería de imágenes este será mostrado en tamaño grande con un carrusel, incorpora automáticamente post relacionados, y se ha habilitado un menu de administración

== Frequently Asked Questions ==
¿Cómo funciona la galería de imágenes?
Ingresar a registrar un nuevo post, presionar el botón insertar objeto, seleccionar la plantilla galería, elegir las imágenes que formaran parte de la galería y publicar. Si esta activo elplug in hacer click sobre las imágenes y se mostrara el efecto para ver las imágenes
¿Cómo configurar el widget de información de bolivia?
Ingresar a administración a apariencia , seleccion el widget y colo que los parametros de ciudad, numero de noticias.
